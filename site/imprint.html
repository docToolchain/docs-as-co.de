Datenschutz · Privacy

This website is intentionally privacy-friendly. It is a static site and sets no cookies. We do not run advertising or third-party tracking beyond what is listed below.

Reichweitenmessung (Analytics)

We use GoatCounter for anonymous, aggregated visitor statistics. GoatCounter is cookieless and does not collect personally identifiable information; it does not build profiles and does not share data with third parties.

Hosting & Server-Logs

The site is hosted on GitHub Pages (GitHub, Inc. / Microsoft). For technical operation and security, the host may process standard server access logs, which can include the visitor’s IP address, the requested page, time, and user agent.

Web Fonts

All fonts are self-hosted on this site’s own server. No font files are loaded from Google Fonts or any other third party, so no visitor IP address is transmitted to an external font service.

External links

This site links to external websites (for example doctoolchain.org, arc42.org, github.com). We are not responsible for the content or privacy practices of those sites.
