# Archived: legacy Jekyll site

This folder holds the previous **Jekyll + AsciiDoc** version of docs-as-co.de
(blog posts from 2016–2017, the minimal-mistakes theme, the Gradle/Docker build,
and the htmlSanityCheck setup).

It was retired in June 2026 when the site was relaunched as a lean, static
HTML/CSS funnel site living in [`/site`](../../site). The new site is deployed via
GitHub Pages — see [`.github/workflows/pages.yml`](../../.github/workflows/pages.yml).

The content is kept here (and in the git history) for reference only. It is **not
built or deployed** anymore. Still-relevant external links were carried over into
the new `site/resources.html`.
